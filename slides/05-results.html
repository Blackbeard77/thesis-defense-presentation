Economic Evaluation

Estimated Operational Costs

Cloud-Based (GPT-4o)

Per Player: ~$3.77

• ~62,767 output tokens @ $10/1M
• ~1,255,340 input tokens @ $2.50/1M
• Equals 15% of game price ($24.81 avg)

Local LLM (Gemma 3 1B)

Per Player: $0.00

• One-time hardware investment
• No recurring API costs
• Higher minimum system requirements
• 100% offline capability

At scale, cloud costs become unsustainable: 10,000 players = $37,700 in API fees alone.
